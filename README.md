## 扫码点餐

简单的扫码点餐系统, 客户端可以扫描二维码进开始点餐, 商户可以管理订单状态, 管理菜品, 修改桌面,
多人一起点餐菜品同步显示功能

## 技术栈

前端主要使用了 React, React-router, AntDesign.  全部采用 Hook 组件完成. 
顾客下单页使用了 Redux
桌面管理页使用了 useReducer

后端主要使用了Express, SQLite.

前后端使用了 WebSocket 完成实施点餐功能
